
# Repository Pattern

- Repository Pattern genel bir tasarım yaklaşımıdır.
- Amacımız ve  erişim katmanını uygulamanın geri kalanlarından ayırmaktır. Yani  iş veya servis katmanı dogrudan EF Core, Sql yada DbContext ile uğraşmaz araya repository girer 
### Örneğin

 Controller -> Service -> Repository -> DbContext
 ---

 ```csharp
public interface IBookRepository
{
    IEnumerable<Book> GetAll();
    Book GetById(int id);
    void Create(Book book);
    void Update(Book book);
    void Delete(Book book);
}
```
Burada **Book** için özel bir repository vardır. Bu, klasik repository pattern örneğidir.
